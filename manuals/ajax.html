Ajax

Get

Dats how we load html content using AJAX:

_tjs.ajax.get(selector, URL);

selector - a block you'll append loaded HTML

URL - URL of the page you wanna get content from

Here you can see some examples:

_tjs.ajax.get('#code-example-ajax', '/examples/ajax.txt');

As you see - get function loads content of this page: /examples/ajax.txt
